## Description

## Instruction
- To run project in local, please follow these steps:
- Step 1:
     - Install nodeJS and expo go
- Step 2:
     - Open terminal in IDE or cmd and run the command: npm install
- Step 3:
     - Run the command: npm start
     - Use expo app scan QR code in console.
